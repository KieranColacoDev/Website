Projects

Evear Hearing Website

I created a professional and mobile-responsive website for Evear Hearing using Wix, emphasizing functionality and user experience. The Home page welcomes visitors with an introduction to their mission and services. The Hearing Aids page showcases advanced devices, while the Services page offers a detailed overview of all their services. The Location page shares clinic details and utilizes the Google Maps API, and the Book Appointment page simplifies scheduling for clients.

Canaray Website

I led the redesign of Canaray's key web pages, including the landing page, services page, and location page. For the landing page, I focused on creating an engaging and visually appealing introduction that captured the visitors' attention. For the services page, I created a new design to display the services in a more user-friendly manner. For the location page, I utilized the Google Maps API to integrate an interactive map into the location page, allowing users to easily navigate to various locations.

Dash Chat Application

I led the design and development of a dynamic mobile chat application for Canaray employees. I used Figma to create detailed wireframes, which guided the application's layout and design. For the front end, I used Vue.js and the Ionic framework to ensure a responsive and intuitive user experience. Additionally, I orchestrated the integration of multiple APIs to seamlessly retrieve user data, enhancing the application's back end functionality and performance.

Google Ad Campaigns

I executed impactful Google Ad campaigns for Inforica, targeting their key services: Digital Transformation, Cloud Computing, Data & Analytics, Cybersecurity, Fractional Resourcing, and Microsoft Dynamics 365. By utilizing SEO tools like Google Keyword Planner and SEMrush, I conducted in-depth keyword research to optimize ad copy and ensure precise targeting. Continuous performance tracking through Google Analytics allowed me to make data-driven adjustments.
